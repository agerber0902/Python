## Player Stats Predictor
Leverages BeautifulSoup and requests libraries to scrape [sports reference player pages](https://www.pro-football-reference.com/players/) by input url.
User will input a player url, and the application will call that url via requests, and parse the web page using BeautifulSoup
Application displays various output data regarding player data and trends.

Application was created in Spyder 6
